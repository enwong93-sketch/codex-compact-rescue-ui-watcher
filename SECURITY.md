# Security

This project automates the local Codex Desktop UI on Windows. It does not collect credentials, send telemetry, or make network requests.

## Reporting

If you find a security issue, open a GitHub issue with reproduction details and mark it clearly as security-related.

## Safety Notes

- Review scripts before running them.
- Keep the watcher scoped to your own Windows user session.
- Do not run this on a locked, shared, or unattended machine where UI automation could interact with the wrong window.
